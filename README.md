# RuletaPataleta
Con esta aplicación te echarás unas risas.
Cuando no sabes qué hacer para el próximo partido, quién se queda en el banquillo, quién será la próxima colocadora, gira la ruteta y verás qué sorpresas.

Elige las fotos de tus jugadoras (entre 2 y 16), pon sus nombres en los archivos de las fotos, y deja todo en manos de la suerte.
